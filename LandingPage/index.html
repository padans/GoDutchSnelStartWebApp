GoDutch → SnelStart
Bankafschriften automatisch importeren
  • ✓ Transacties ophalen per bankrekening (IBAN) en datumperiode
  • ✓ Export naar SnelStart in MT940 of CAMT053 formaat
  • ✓ Automatische berekening van openings- en eindsaldo per exportperiode
  • ✓ Auto-sync via achtergrondworker met instelbaar interval per bankrekening
  • ✓ Handmatige resync vanaf een zelfgekozen datum
  • ✓ Webhook-ontvangst: SnelStart-import getriggerd door GoDutch-events
  • ✓ Retry-mechanisme bij tijdelijke SnelStart-uploadfouten (exponential backoff)
  • ✓ Elke sync-run vastgelegd met status, triggerobron en statistieken
  • ✓ Syncperiode automatisch bepaald op basis van laatste succesvolle run
  • ✓ GoDutch-rekeningen opvragen rechtstreeks vanuit de GoDutch API

Hoe werkt het

In drie stappen aan de slag

Na aanmelding kunt u direct starten. Geen installatie, geen technische kennis vereist.

1

Aanmelden

Vul uw bedrijfsgegevens in en kies de gewenste module(s). Uw proefperiode van 7 dagen start direct.

2

Koppelen

Voer uw GoDutch- en/of SnelStart API-gegevens in via het beveiligde portaal. Bankrekeningen worden automatisch herkend.

3

Automatisch verwerken

De koppeling werkt volledig automatisch op het door u ingestelde interval.
